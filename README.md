# movie-cards-mock-with-react-native

A simple mock for movie and character cards using react-native and tailwindcss.

## Libraries used

- react-native
- tailwind-rn
- react-navigation
- react-navigation-stack
- expo-linear-gradient

## Installation

` npm install -i`
